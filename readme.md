# 安装

可以使用 node 全局安装

```
npm install -g @star-one/git-commit
```

如果不想全局安装也可以在项目目录下安装

```
npm install -D @star-one/git-commit
```

# 使用

> **前置条件：需要手动暂存想要提交的文件**

如果全局安装使用命令`gec`进入 git commit 流程
如果是项目目录下安装使用命令`npx gec`进入 git commit 流程
